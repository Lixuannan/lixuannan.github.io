Ubuntu 24.10 的 6 个新功能

Ubuntu 24.10 的 6 个新功能

本文翻译自 https://news.itsfoss.com/ubuntu-24-10/

每年我们都会看到 Ubuntu 发布一个新的临时版本，这些版本会由 Canonical 提供为期九个月的支持，之后该版本将不再获得任何新更新或关键的安全补丁。通常，喜欢稳定体验的人会选择长期支持（LTS）版本的 Ubuntu，该版本维护的时间更长，最长可达 12 年，具体取决于发布的版本。如今，在 Ubuntu 24.04 LTS 发布几个月后，我们迎来了 Ubuntu 24.10 临时版本，代号为 “Oracular Oriole”。

但这还不是全部；这个版本还标志着 Ubuntu 20 周年纪念日，Canonical 为此特别设计了主题墙纸，并配上了一种非常空灵的启动音效，使这个版本独具特色。继续阅读，了解 Ubuntu 24.10 “Oracular Oriole” 的更多新功能。 👇

1. Linux Kernel 6.11

Ubuntu 24.10 由 Linux kernel 6.11 提供支持，该版本带来了一些有趣的硬件升级。包括 针对即将发布的 AMD RDNA4 GPU 的补丁，对 Intel Battlemage GPU 的初步支持，简化了 Intel CPU 的 HWMon 代码，并且更好地支持 LoongArch CPU（中国入狂喜）架构。

2. GNOME 47

Ubuntu 24.10 的第二大亮点无疑是 GNOME 47，它带来了许多酷炫的功能，如 支持颜色强调（顺便说一下，Ubuntu 已经具备了这个功能）。此外，还有 许多更新的 GNOME 应用，如 GNOME Terminal 3.54.0、System Monitor 47、Clocks 46、Calculator 47 和 Eye of GNOME 47。此外，还新增了通过 壁纸竞赛 选出的新壁纸。

3. 更好的用户体验

文件管理器应用 Nautilus（文件）有了许多改进，最显著的是重新设计了侧边栏，现在 下部有可移除的书签。另外，任何内部存储设备现在默认显示在侧边栏，文件传输对话框已重新组织，“其他位置”页面被新“网络”页面取代。

Ubuntu 24.10 的 Dock 引入了 对正在后台更新的 Snap 应用进度条的支持，此外，右键点击任何应用时会显示更新的上下文菜单。

4. 应用中心调整

基于 Flutter 的应用中心也得到了关注，“管理”页面允许轻松卸载 Snap 应用，并新增了一个名为“艺术与设计”的分类，展示了 GIMP、Inkscape 等流行应用。

5. 安全升级

Ubuntu 24.10 引入了一个专用的安全中心，目前仅包含 Snap 应用的实验性 App 权限设置，并为 OpenVEX 和 OSV 格式的漏洞报告提供支持，还改进了 PPA 的安全性，APT 现在要求 PPA 和其他仓库具有更强的签名。

6. 面向开发者的更新和软件包更新

Ubuntu 24.10 还包含了最新的 Python、Java、Go、C、C++ 和 .Net 版本，以及 Rust 1.80、Valkey 和 OpenJDK 21 等开发者工具和软件包的更新。

如需了解更多信息，建议阅读 官方博客。

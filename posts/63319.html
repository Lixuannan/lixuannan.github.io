构建超强 OI 系统之开发环境

环境的配置

常用的环境有 Python、C/C++、LaTeX 等等。下面介绍以下几种环境的安装方法。

Python

直接在终端运行 sudo pacman -S python 即可，如果你要安装 Python2，可以使用 sudo pacman -S python2。

C/C++

对于大部分的编译器直接使用 pacman 安装即可，比如 gcc 可以直接使用 pacman -S gcc 安装即可。

代码编写软件

下面简单介绍常用的软件，包含 Jetbrains 软件、VSCode 以及人见人爱，花间花开的 VIM。

Jetbrains 软件（IDEA, CLion, Pycharm 等）

安装 Jetbrains 的软件略微有一些麻烦，首先需要安装一个包管理器：flatpak。使用命令 sudo pacman -S flatpak 即可。

然后我们就开始安装了，使用 flatpak install clion 或者其他 IDE 的名字安装即可。注意你需要一直同意他的操作。

VSCode

使用命令 yay -S visual-studio-code-bin 安装即可，安装后与其他操作系统版本的配置方法相同。

VIM

需要注意的是，在 Arch Linux 上存在着一个特别的现象，如果你是用的是安装 gVIM 的时候附带的 VIM，那么你拥有的特性是比直接安装 VIM 要多的。这就导致我们应该安装 gVIM。

使用 sudo pacman -S gvim 来安装 gVIM。然后你可以直接抄作业使用我的配置文件：https://github.com/Lixuannan/vimrc/blob/main/.vimrc。你需要打开一个终端，然后在你的用户目录下创建一个名叫 .vimrc 的文件然后把链接里面的内容粘贴进去。

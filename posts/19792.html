零基础入门 OI 第二节——编码前的准备

零基础入门 OI 第二节——编码前的准备

在开始学习有关编码的内容之前我们应该获得一套称手的或者说可用的编程设备。下面就介绍一下硬件与软件方面的准备。

硬件

对于编程来说，一台任意品牌的能够正常开机的电脑应该都是可以满足需求的。但是老爷机可能无法运行所需的环境。可以通过打开 https://www.bilibili.com 来验证电脑的性能，如果能够正常的浏览视频、发弹幕，想必是没有问题。

软件

下面简单介绍一下常见的开发环境。

RedPandaCpp

对于刚入门的 OI 选手，选用 RedPandaCpp 是一个非常不错的选择，可以前往 官网 下载安装。下载时注意不要选择具有 No-Compiler 即无编译器标签的安装包。下面简单介绍一下使用方法。

可以看到这是一个清晰的编辑页面。你可以在有行号的编辑区编辑你的代码，然后使用 <F11> 键编译运行。详细的操作指引请看官方的使用帮助。

VSCode

VSCode，全称 Visual Studio Code，是由微软公司开发的一款“轻量级”代码编辑器。你可以访问微软的官方网站获取 VSCode 的最新安装包。安装后你还需要安装编译器，你可以前往搜索引擎自行搜索你对应系统的 C++ 编译器安装方法，Google is your friends.

对于 Windows，你可以下载预编译的 MinGW，前往 GitHub Release 下载你对应架构的编译器，一般来说，你需要下载 x86_64 开头的压缩包。下载压缩包之后解压并将解压出来的文件夹中的 bin 文件夹所在的目录加入环境变量中。具体操作：

  • 复制文件夹目录，可能是 C:\User\Admin\Desktop\mingw\bin\。

  • 右键单击此电脑-属性-高级系统设置-环境变量

  • 然后编辑系统的 PATH 环境变量，新建，然后粘贴刚刚复制的路径，保存即可。

然后你需要在安装 CPH 插件，这样你就可以快速的运行代码了。

具体使用方法自行 Google。

VIM

VIM 只推荐在非 Windows 系统上面使用，具体安装方法可以参考我以前写的一篇文章：构建超强 OI 系统之开发环境。

那么你已经学会如何给自己准备一个开发环境啦！
